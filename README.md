# Facial-Emotion-Detection-Using-XAI
This project implements a real-time facial emotion detection system using computer vision and machine learning, enhanced with Explainable AI (XAI) techniques to improve model transparency.
The system captures live video input through a webcam, detects human faces using OpenCV, and classifies emotions such as happy, sad, angry, and neutral using a trained model. To make predictions more interpretable, XAI methods like LIME or Grad-CAM are applied to highlight the facial regions that contribute most to the model’s decision.

Key Features:
Real-time emotion detection using webcam

Face detection and preprocessing with OpenCV

Emotion classification using ML/DL models

Explainability using LIME / Grad-CAM

Visual output showing prediction insights

Tech Stack:
Python, OpenCV, NumPy, Pandas, TensorFlow/Keras/PyTorch, LIME/Grad-CAM

Applications:
This system can be used in human-computer interaction, mental health monitoring, smart surveillance, and emotion-aware AI systems.
